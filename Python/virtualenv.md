# Python Virtual Enviornments

## Setup
```
## 'venv' is standard naming convention; this is project name ##
$cd to project folder /tmp or wherever
$virtualenv venv (if you want to use python 3.x)
$virtualenv -p /usr/bin/python2.7 venv
$source venv/bin/activate (to activate the project)
$deactivate (will exit the project)

[docs.python-guide.org] https://docs.python-guide.org/dev/virtualenvs/
```

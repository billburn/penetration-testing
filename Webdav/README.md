# Webdav Testing

## Davtest

```
$davtest -url http://url-of-the-host
```

## Davtest through proxy

```
Open Burp
Setup new proxy listener
Set local listener port *(80) 
Set remote host (10.10.10.15)
Set remote port (80)
```

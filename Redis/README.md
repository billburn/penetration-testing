# Redis Hacking

## Tools
```
redis-cli
```

## Connect to the database
```
redis-cli -h 10.10.10.10
```

## Show all the keys in the DB
```
show *
```

## Get a specific key
```
get <number>
```

## Get information about a server
```
info
```
